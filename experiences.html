Lockheed Martin Internship

July 2023 – September 2023

I developed python applications to support LM Space cybersecurity efforts. My 2 main projects were creating the NocoDB Security Control Analyzer and automatic CMDB updates. For the NocoDB project, I created a python application that analyzes Security Control Applicability using regex and natural language processing for the NIST 800-53 spec. I used REST APIs to interface with NocoDB to create a accessible tool on their intranet. Ultimately, this project led to better management planning and enhanced security engineering efficiency. As for automating CMDB updates, I built scripts to automatically filter and designate devices lacking complete classifications. The project also employed REST APIs to authenticate and update the CMDB. My efforts resulted in improved categorizations, traceability, and therefore security.
